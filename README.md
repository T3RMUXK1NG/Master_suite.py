### Core Features
- **Interactive Menu** — Easy-to-use numbered category navigation
- **Tool Management** — Install, run, and view documentation for each tool
- **Installation Detection** — Shows [✓] or [✗] for each tool's availability
- **Session Tracking** — SQLite database logs tool usage sessions
- **Auto-Environment Setup** — Creates required directories on launch
- **Termux Compatible** — Works on Android via Termux

## 📖 Usage

1. **Launch** the suite with `python3 Master_suite.py`
2. **Select** a category (1-16) from the main menu
3. **Choose** a tool from the category
4. **Manage** the tool: Install, Run, or View Documentation
5. **Type 0** to go back, **17** to exit

### Auto-Created Directories
```
tools/  logs/  results/  wordlists/  payloads/  exploits/
reports/  captures/  downloads/  screenshots/  databases/
backups/  scripts/  binaries/  temp/
```
